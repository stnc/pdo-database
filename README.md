## 3. Select multiple rows
```php
$q = "SELECT * FROM ".$tableName;
$array_expression = $db->rows ( $q );
foreach ( $array_expression as $value ) {
	echo  $value ['name'];
	echo '<br>';
}
```
## 3. Select single row
